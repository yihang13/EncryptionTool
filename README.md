# 加密工具 Encryption Tool V1.0 简介

这是一个一个基于 Win32 API 的 Windows 桌面加密解密工具，使用 **XOR 循环密钥异或 + 自定义编码** 算法，通过图形界面进行操作。

> 宇宙安全声明：
> 本工具只适用于学习、生活和研究使用，请勿用于其他不正当用途。
> 一切因为不正当用途所带来的责任均由使用者承担。

### 加密算法

#### 第一层：XOR 循环异或 (`xor_calc`)

使用**循环密钥**对明文的每个字节依次进行异或运算：

```cpp
cipher[i] = plaintext[i] ^ key[i % key.length()];
```

* 密钥按字节循环使用，覆盖整个明文。
* 异或运算天然可逆：`A ^ K ^ K = A`，因此加密和解密使用同一个函数。

**示例：**

```
明文：Hello
密钥：abcab
异或后：(H^a)(e^b)(l^c)(l^a)(o^b)
```

#### 第二层：Base16 编码 (`to_mi` / `from_mi`)

异或运算产生的字节可能是不可打印字符，为便于显示和复制，将每个字节编码为由 **大写字母 A-P**（共 16 个字母，代表 0-15）组成的两个字符：

**编码**：

```cpp
res += 'A' + (byte / 16);   // 高 4 位 -> A~P
res += 'A' + (byte % 16);   // 低 4 位 -> A~P
```

**解码**：

```cpp
byte = (s[i] - 'A') * 16 + (s[i + 1] - 'A');
```

**示例** ： 

```
字节 0x4F 的编码过程：
0x4F = 79
高 4 位: 79 / 16 = 4  → 'A' + 4 = 'E'
低 4 位: 79 % 16 = 15 → 'A' + 15 = 'P'
结果: "EP"
```

这与 Base16 编码原理一致，只是用 A-P 替代了 0-9A-F。

#### 加密 / 解密完整流程

| 操作 | 流程 |
|------|------|
| **加密** | 明文 → XOR 循环异或 → Base16 编码 → 密文字符串 |
| **解密** | 密文字符串 → Base16 解码 → XOR 循环异或 → 明文 |

由于 XOR 运算的可逆性和 Base16 编码的无损性，解密过程能准确还原原始明文。

### GUI 界面

基于 Win32 API 构建的原生 Windows 窗口，固定大小约 **660×500 像素**。

#### 控件说明

| 控件 | 说明 |
|------|------|
| 密钥输入框 | 单行输入框，输入加密密钥 |
| 内容输入框 | 多行输入框，可输入/粘贴待加密或待解密的文本 |
| 加密按钮 | 执行加密操作 |
| 解密按钮 | 执行解密操作 |
| 结果输出框 | 只读多行框，显示加密/解密结果 |
| 复制结果按钮 | 将结果框中的内容复制到系统剪贴板 |

#### 界面特性

- 字体：**Microsoft YaHei UI**，18pt，ClearType 抗锯齿
- 配色：浅灰背景 (`#F5F7FA`)，白色输入框，深灰色标签文字
- 输入验证：密钥或内容为空时弹出提示框
- 文本编码：内部使用 **UTF-8** 处理所有字符串

### 使用方式

### 编译

使用 g++ 编译（需安装 MinGW-w64 或其他 GCC Windows 工具链）：

```powershell
g++ -O2 Encryption.cpp -o EncryptTool.exe -mwindows -static
```

#### 运行

双击生成的 `EncryptTool.exe`，在界面中进行操作：

1. 输入**密钥**
2. 在**内容**框中输入待加密文本
3. 点击**加密**按钮
4. 结果框中显示密文，点击**复制结果**即可粘贴到其他地方
5. 解密时，输入与加密时相同的密钥，将密文粘贴到内容框，点击**解密**即可还原

### 安全性说明

该工具使用简单的 XOR 异或加密，属于**古典密码**范畴，适用于：

- 防止明文内容被直接阅读（轻度混淆）；
- 在学校等生活化场景中加密通信；
- 因为密文只包含大写字母，所以适合手写；
- 学习加密算法的原理。

**不适用于**需要强密码保护的场景。主要局限：

- XOR 密钥循环复用，在明文较长时易被频率分析攻击。
- 没有认证机制，密文可被篡改。
- 建议重要数据使用 AES、ChaCha20 等标准加密算法。

### 核心函数速查

| 函数 | 参数 | 返回值 | 功能 |
|------|------|--------|------|
| `xor_calc` | `s: string`, `key: string` | `string` | 使用循环密钥对 s 逐字节 XOR |

| `from_mi` | `s: string` | `string` | 将 A-P 字母串解码为二进制字符串 |
| `DoEncrypt` | — | — | GUI 加密处理函数 |
| `DoDecrypt` | — | — | GUI 解密处理函数 |
| `DoCopy` | — | — | 将结果复制到剪贴板 |
